======================

FpsControl plugin
by NeatyAssets AS
Version: 1.0

======================

This plugin is a simple FPS controller for Unity3D.
It addresses the issue when Unity Editor framerate is not capped by default.
It may cause high CPU/GPU usage, noise and overheat when it's not necessary, 
ex. when you are running on battery and carry your machine around

NOTE: This is an Editor Only plugin

How to use this plugin:
 1. Import this package into your Unity project.
 2. Go to Unity top toolbar, and check Tools -> Fps Control -> Enable 
 2. Select desired frame rate, default is Unlimited
 3. Select On battery frame rate if required, default is Identical to desired frame rate.
 3. To disable plugin -> Tools -> Fps Control -> Enable
